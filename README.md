# Global Bond Market Map

A live map of global sovereign bond markets, built to visualize yield levels and cross-market moves in a single view.

## Overview

The Global Bond Market Map is part of the BondStats platform.

It is designed to make global bond markets easier to read by showing how sovereign yields are distributed across countries and how they move relative to each other.

Instead of scanning individual charts, the map provides a structured visual overview of:

- Yield levels across markets  
- Daily yield changes  
- Cross-market divergence  
- Regional patterns  

## Features

- Global sovereign bond map (ISO-based coverage)
- Toggle between:
  - Yield levels  
  - Daily moves (basis points)  
- Hover interaction:
  - Country  
  - Yield  
  - Daily move  
- Top movers panel  
- Highest yield ranking  
- Live market table  
- Share-ready view
